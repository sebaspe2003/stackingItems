Proyecto: stackingItems
========================

Simulador de torre de tazas para el problema del maraton ICPC 2025.

Este proyecto permite simular una situacion donde se apilan tazas en una torre.
Cada taza tiene un numero y puede tener una tapa. Las tazas se apilan verticalmente
y las tapas se colocan sobre las tazas.

Clases principales:
- StackingItems: Clase principal del simulador
- Tower: Representa la torre donde se apilan las tazas
- Cup: Representa una taza individual
- Lid: Representa una tapa

Clases de soporte (del proyecto shapes):
- Canvas: Lienzo para dibujar
- Rectangle: Figura rectangular (usada para tazas)
- Circle: Figura circular (usada para tapas)
- Triangle: Figura triangular

Autores: Nicolas Prieto y Sebastian Peña
Version: 2.0
Fecha: 2025
